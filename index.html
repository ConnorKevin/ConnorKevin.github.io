RobotHusky-Field Agriculture Robot
As part of the robot project, we use a classification algorithm based on VGG-16 to identify the maize and grass. We collected the pictures of maize in the field and augmented the data set with flip, rotate and scale, tried several ways, and finally decided to use the result with 89% precision based on the VGGNet.
